- HTML5
- CSS3 (custom properties, animations, grid, flexbox)
- Vanilla JavaScript (scroll reveal, custom cursor, intersection observer)
- Google Fonts (Playfair Display, Plus Jakarta Sans, Crimson Pro)

## Run Locally

No build step needed. Just open `index.html` in your browser.

```bash
git clone https://github.com/abhishekjha462000/abhishekjha462000.github.io
cd abhishekjha462000.github.io
open index.html
```
